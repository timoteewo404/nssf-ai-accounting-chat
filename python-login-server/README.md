# Simple Python Login Server

This is a tiny Python login server using only the standard library.

## Run

1. Open a terminal in this folder:
   ```powershell
   cd "c:\Users\RHITEX\OneDrive\Desktop\revision\python-login-server"
   python server.py
   ```

2. Open the browser at:
   ```
   http://localhost:8000/login
   ```
